## 💡 **EXAMPLE: WHAT THE CHATBOT DOES**

### Input
```
Student: "My risk score is 65 and I'm worried. What do I do?"
```

### Processing
1. ✅ Validate student exists
2. ✅ Fetch: Risk score, attendance, marks, pending assignments
3. ✅ Rotate API key (automatic)
4. ✅ Build context-aware prompt
5. ✅ Call Gemini API
6. ✅ Generate suggestions
7. ✅ Save to database

### Output
```json
{
  "reply": "I see you have a High risk score of 65/100. 
    Here's your action plan:
    
    1. ATTENDANCE (Most Important): You're at 72%, need 75%
       - Attend all remaining classes
       
    2. MARKS (Important): Your average is 58%
       - Focus on weak subjects
       
    3. ASSIGNMENTS (Do This Week): You have 3 pending
       - Complete the highest priority first
    
    You're capable of improvement! Start TODAY. ✅",
    
  "suggestions": ["Improve attendance", "Complete assignments", "Study weak subjects"],
  
  "riskInsights": {
    "currentRiskScore": 65,
    "category": "High",
    "recommendations": [
      "URGENT: Attend all remaining classes...",
      "Focus on strengthening weak subjects...",
      "Complete pending assignments immediately..."
    ]
  }
}
```

## 🏗️ **SYSTEM ARCHITECTURE**

```
Frontend (React)
     ↓
REST API (/api/chat)
     ↓
Server Actions (Next.js)
     ↓
Fetch Student Context (Supabase)
     ↓
Build AI Prompt (with context)
     ↓
Chatbot Service
     ↓
Get Rotated API Key
     ↓
Call Gemini API
     ↓
Generate Response
     ↓
Save to Database
     ↓
Return to UI
     ↓
Display with Animations
```
